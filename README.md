# Smash!!!! 🎾

ソフトテニス ダブルスマッチ用
**観戦者向けスコアボード Webアプリ**

---

## 📌 このアプリについて

**Smash!!!!** は、
部活動やスポーツ観戦をより楽しくするために作られた
無料のソフトテニス ダブルスマッチ用スコアボード Webアプリです。

特にソフトテニスは、

* ルールが比較的複雑
* スコアボードが設置されていない公式試合も多い

といった理由から、
観戦者が試合の流れや得点状況を把握しづらい場面があります。

本アプリでは、
**観戦者自身が得点を記録しながら試合を追う**ことで、
より深く試合を理解し、応援を楽しめることを目的としています。

---

## ⚠️ ご利用にあたっての注意

* 本アプリは **公式記録用ではありません**
* 観戦・応援を目的とした補助ツールです
* 記録の正確性を保証するものではありません

大会・公式記録には、
必ず主催者・審判の指示に従ってください。

---

## 🎮 主な機能

* ダブルス専用スコア管理
* 観戦位置に合わせたコート回転
* サーバー表示・管理
* ポイントの追加・巻き戻し
* 試合画面のカスタマイズ
* お気に入り設定の保存（最大5件）

※ 操作はすべて **タップ中心** で、
保護者・観戦者の方でも直感的に使える設計を目指しています。

---

## 🖥️ 使い方（概要）

1. 設定画面で試合数などを設定
2. 観戦位置に合わせてコートを回転
3. 選手・サーバーを設定
4. 試合開始後、ポイントをタップで記録

詳しい操作方法は、
アプリ内の **インフォメーション（i）** をご覧ください。
