応援・サポート 🙌

このソフトテニス ダブルスマッチ スコアボードアプリ Smash!!!! は、
部活動やスポーツ観戦をより楽しくするために作られた無料のWebアプリです。
特にソフトテニスは、ルールが比較的複雑であり、
スコアボードが設置されていない公式試合も多く、
観戦者が試合の流れや得点状況を把握しづらい場面があります。
本アプリでは、観戦者自身が得点を記録しながら試合を追うことで、
より深く試合を理解し、応援を楽しめることを目的としています。

もし「便利だな」「使ってよかった」と感じていただけましたら、
開発の励みとして、任意で支援していただけると嬉しいです。
